# GTAV-DMA-v2
GTAV DMA intended for singleplayer with features from two different other things.

Merged https://github.com/chuckmorris420/GTAV-DMA/tree/main with https://github.com/YimMenu/YimMenuV2 for GTAV Enhanced Edition

Not all features are present due to script constraints.

The important features (slots and hesit recovery) works but may be glitches or issues.


how to use:
first you need to open GTAV without BE and inject YimMenu

Get your tunables.bin files from appdata/roaming/yimmenu

open the sln and build it.

now moves the files from the "move to your binary folder" to your x64 folder with the exe, alongside the tunables. (tunables isnt needed unless you want to use RP multipler or some other stuff which is WIP)

now you can launch your game with BE on and it will work

ive tested on BE, ive been kicked a few times and asked to restart the game but I was being very blatant with recovery as part of my test
